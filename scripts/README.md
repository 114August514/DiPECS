# Scripts

本目录存放**开发者日常可能运行的脚本**，按用途分子目录。

## 子目录

- `ci/` — CI 专用脚本，由 `.github/workflows/` 调用；本地也可手动运行。
- `dev/` — 开发环境、本地检查、git hooks。
- `android/` — Android 模拟器/真机操作。
- `docs/` — 文档/学术报告构建。

## 与 `tools/` 的区别

- `scripts/`：命令式辅助脚本，设置环境、跑检查、操作设备、构建文档。
- `tools/`：需要真机/模拟器或特定输入才能产出**结构化数据集/产物**的测量与生成工具
  （见 `tools/README.md`）。
